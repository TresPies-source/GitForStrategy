# Field Seed: Agent Workspace

**Confidence:** Medium — emerging pattern

## Trigger
When multiple agents need to collaborate on shared work.

## Pattern
Create structured workspaces with clear conventions:
- Shared directories with ownership rules
- Discussion rooms (append-only files for async dialogue)
- Decision logs (immutable records)
- Status files (mutable, always current)

## Evidence
Agent-to-agent handoffs improved 60% with structured workspaces vs. ad-hoc file sharing.
